# 宝可梦属性克制交互图

这是一个用于展示宝可梦 18 种属性克制关系的单页面应用。

页面主要包含两部分：

- 左侧信息区：展示当前选中的防御属性，以及对应的被攻击倍率汇总结果
- 右侧属性卡片区：展示 18 种属性各自的克制关系

当前页面功能包括：

- 展示 18 种属性的克制信息
- 支持选择 1 个或 2 个防御属性
- 自动计算双属性组合下的攻击倍率结果
- 将结果按 `4x`、`2x`、`1x`、`0.5x`、`0.25x`、`0x` 分类展示
- 使用不同底色区分强克、弱克、免疫和一般效果
- 支持折叠信息说明区域
- 支持响应式布局，适配不同屏幕尺寸

当前页面基于第六世代起的 18 属性克制规则实现。
